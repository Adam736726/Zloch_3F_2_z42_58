Pętla for: wykonuje się określoną ilość razy.
Konstrukcja: for (zainicjowanie_zmiennej; warunek; zmiana) { kod_petli }

Pętla while (z kontrolowanym wejściem): wykonuje się dopóki warunek jest prawdziwy. Sprawdza warunek przed wykonaniem kodu.
Konstrukcja: while (warunek) { kod_petli }

Pętla do...while (z kontrolowanym wyjściem): wykonuje kod przynajmniej jeden raz, a warunek sprawdzany jest dopiero na końcu.
Konstrukcja: do { kod_petli } while (warunek)
