1. Jak określić właściwość obiektu?

Sposób 1:
obiekt.wlasciwosc = wartosc;
np. auto.marka="Opel";

Sposób 2:
obiekt["wlasciwosc"] = wartosc;
np. auto["rok"]=2020;

2. Jak odwołać się do pól i metod?

Poprzez nazwę obiektu i kropkę.
np. samochod.kolor
np. samochod.pokaz()

3. Cechy konstruktora

Uruchamia się podczas tworzenia obiektu
Ma nazwę jak klasa
Nie zwraca wartości
Może przyjmować argumenty

4. Przykład obiektu

var osoba = {
nazwisko:'Kowalski',
imie:'Jan',
zawod:'piekarz',
wyswietl:function(){
document.write(this.nazwisko + " " + this.imie);
}
}

5. Znaczenie this

this odnosi się do aktualnego obiektu.

6. Konstruktor w JS

function klient(nazwisko,imie,zawod){
this.nazwisko=nazwisko;
this.imie=imie;
this.zawod=zawod;
}

7. Słowo new

var osoba1 = new klient("Kowalski","Jan","kierowca");

8. Prototype

Prototype pozwala dodawać nowe metody i właściwości do obiektów.
